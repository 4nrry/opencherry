## [0.0.1] - 2026-06-22

### Added

- Initial public alpha of OpenCherry: an Ubuntu/GNOME-first, multi-repo ×
  multi-agent control tower for AI-assisted coding, built on Tauri 2.
- Multi-repo sidebar with per-repo Git status, grouped diffs (conflicted,
  staged, unstaged, untracked), stage/unstage, commit, publish, and sync.
- Running-agent detection (Claude Code, OpenCode, Codex, Gemini CLI, Aider, …).
- Linux packages: `.deb`, `.AppImage`, and `.rpm`.
- macOS disk images (`.dmg`) for Apple Silicon and Intel.
- Windows NSIS installer (`.exe`).
- One-command Linux install script (`scripts/install.sh`) and SHA-256 checksums
  attached to every release artifact.
